Solar Panel Visualization

Project Description

This project utilizes ArcGIS JavaScript API to create an interactive 2D map visualizing solar panel installations by year and land cover types. The map includes built-in widgets like legends and layer lists, offering users a seamless experience for exploring solar panel data.

Key Features:

  • Interactive 2D map with real-time data visualization.
  • Land cover analysis using feature layers and media widgets.
  • Year-based unique-value renderers for clear categorization.

Skills Used: JavaScript, ArcGIS Maps SDK, Data Visualization, Interactive Mapping
