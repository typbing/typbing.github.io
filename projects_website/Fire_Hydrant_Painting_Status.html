Fire Hydrant Painting Status

Project Description

This web application supports the Fire Department by monitoring the painting status of fire hydrants across the city. Built using ArcGIS Web AppBuilder, it helps field crews and staff track which hydrants need repainting and which are completed.

Key Features:

  • Map-based interface for hydrant inspection updates.
  • Visual status tracking by color-coded symbology.
  • Built on ArcGIS Web AppBuilder for rapid deployment.

Skills Used: ArcGIS Web AppBuilder, Public Safety Mapping, Utility Asset Management
