Sign Data Collection – Field Map

Project Description

This project utilizes ArcGIS Field Maps to support the inspection and mapping of traffic signs throughout the city. Data is collected in the field and synced to city systems for infrastructure planning and maintenance.

The system records address, district, sign type, orientation, and color for each asset. Helping to improve field data accuracy and enhances coordination across departments.

Skills Used: ArcGIS Field Maps, Mobile Data Collection, Asset Management, Sign Inventory
