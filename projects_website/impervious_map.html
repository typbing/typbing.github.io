NLCD Impervious Surface Map

Project Description

This project uses Leaflet.js to create an interactive map that visualizes impervious surface data from the National Land Cover Database (NLCD) for the years 2001 and 2021. The map includes a side-by-side layer comparison tool, geocoding capabilities, and measurement tools for enhanced analysis.

Key Features:

  • Visualization of NLCD impervious surface data for 2001 and 2021.
  • Side-by-side comparison of map layers.
  • Geocoding, drawing, and measurement tools for advanced exploration.

Skills Used: JavaScript, Leaflet.js, Spatial Analysis, Interactive Mapping
