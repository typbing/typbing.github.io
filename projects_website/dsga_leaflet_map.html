DSGA Leaflet Map Project

Project Description

The Designated Shellfish Growing Area (DSGA) datalayer was compiled by the Department of Fish and Game's (DFG) Division of Marine Fisheries (DMF). The Shellfish Project designated 302 growing areas in Massachusetts and created this interactive web map to visualize these areas using Leaflet.js.

Key Features:

  • Visualization of 302 designated shellfish growing areas in Massachusetts.
  • Interactive exploration of the DSGA datalayer using Leaflet.js.
  • Color-coded polygons based on shellfish classification.

Skills Used: JavaScript, Leaflet.js, R Markdown, Interactive Web Mapping
