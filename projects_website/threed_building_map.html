ThreeD-BuildingMap

Project Description

The ThreeD-BuildingMap project uses MapLibre GL JS to create an interactive 3D map. Buildings are visualized with height attributes, and dynamic controls allow for seamless navigation and layer toggling.

The project integrates Carto's Dark Matter basemap and showcases both raster and vector layers. This visualization aids in urban analysis and GIS modeling.

Skills Demonstrated

  • 3D Visualization using MapLibre GL JS
  • Dynamic layer controls with toggle functionality
  • Data integration from OpenMapTiles and Esri World Imagery
  • Advanced JavaScript for GIS visualization
