Fire Incident ArcGIS Dashboard

Project Description

This ArcGIS Dashboard visualizes fire incident data to support public safety analysis and emergency response planning. It provides interactive maps, real-time metrics, and filterable charts for better decision-making.

Key Features:

  • Live dashboard with map and statistical widgets.
  • Filtering by date, location, and incident type.
  • Supports fire department planning and reporting.

Skills Used: ArcGIS Online, Dashboard Configuration, Data Visualization, Public Safety GIS
