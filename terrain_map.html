Terrain Map Project

Project Description

This project showcases an interactive 3D Terrain Map built using MapLibre GL JS. It visualizes detailed terrain and satellite data, offering an immersive experience for analyzing topography.

Key Features:

  • 3D terrain visualization with adjustable exaggeration.
  • Hillshade and satellite imagery layers with customizable visibility.
  • Interactive controls for navigation and exploration.

Skills Used: JavaScript, MapLibre GL JS, Terrain Visualization, Interactive Mapping
