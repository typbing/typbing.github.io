Housing Mortgage Map Project

Project Description

This project uses a Shiny App to create an interactive Housing Mortgage Map. It allows users to explore mortgage data across the United States by state and loan amount range.

Key Features:

  • Interactive visualization of housing mortgage data.
  • Dynamic filters for state and loan amount ranges.
  • Real-time updates to map and insights based on user inputs.

Skills Used: R, Shiny, Interactive Mapping, Data Analysis
